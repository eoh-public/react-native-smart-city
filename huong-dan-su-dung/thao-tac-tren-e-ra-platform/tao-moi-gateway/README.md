# Tạo mới gateway

Nhập các thông tin cần có cho Gateway, rồi bấm vào nút **Bước kế tiếp**

* Tên Gateway
* Chọn loại bo mạch hỗ trợ, ở đây người dùng sẽ chọn SP32
* Chọn địa điểm đã tạo hoặc tạo điạ điểm phù hợp
